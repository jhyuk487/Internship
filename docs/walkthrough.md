# Project Walkthrough (Summary of Changes)

이 문서는 최근 진행된 주요 개발 작업과 변경 사항을 요약합니다.

## 1. 데이터 스키마 및 DB 연동 최적화
- **필드명 통일**: `student_id` -> `user_id`, `password` -> `user_password` 등 `data_sets`의 실제 키값과 코드를 일치시켰습니다.
- **직렬화 오류 해결**: MongoDB의 `PydanticObjectId`가 JSON으로 변환되지 않는 문제를 해결하기 위해, 응답 시 해당 필드를 제외하는 로직을 추가했습니다.

## 2. 세션 및 유저 인터페이스(UI) 개선
- **세션 유지 (Auto-Login)**: JWT 토큰을 활용한 `/auth/me` 엔드포인트를 구현하여 새로고침 시에도 로그인이 유지되도록 했습니다.
- **프로필 동적 갱신**: 로그인한 사용자의 이름과 전공을 사이드바에 즉시 반영하고, 이름에 기반한 이니셜 아바타를 생성합니다.

## 3. 안정성 확보
- **AI 컴포넌트 격리**: 네트워크 타임아웃을 유발하던 Hugging Face 모델 로드 로직을 주석 처리하여 서버 실행의 안정성을 확보했습니다.

## 4. 기술 문서화 완료
- `db.md`, `frontend.md`, `login.md`, `ai.md` 등 주요 기능별 가이드를 작성하여 프로젝트 유지보수성을 높였습니다.
