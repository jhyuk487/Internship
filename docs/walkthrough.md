# Project Walkthrough (Summary of Changes)

이 문서는 최근 진행된 주요 개발 작업과 변경 사항을 요약합니다.

## 1. AI 인프라 고도화
- **SDK 마이그레이션**: `google.generativeai`에서 최신 `google-genai` 라이브러리로 전환하여 최신 Gemini 2.0 모델 지원을 확보했습니다.
- **모델 업그레이드**: `gemini-2.0-flash` 모델을 적용하여 응답 속도와 지능을 향상시켰습니다.

## 2. 인증 및 보안 강화
- **비밀번호 찾기 (Password Recovery)**: 학번과 이메일을 활용한 보안 인증을 통해 비밀번호를 조회할 수 있는 기능을 추가했습니다. (In-place UI 적용)
- **게스트 채팅 제한**: 비로그인 사용자의 입력을 차단하고 안내 문구를 표시하여 보안을 강화했습니다.
- **로그아웃 초기화**: 로그아웃 시 대화 내용과 히스토리를 즉각 삭제하여 공용 PC 등에서의 개인정보 노출을 방지했습니다.

## 3. 사용자 경험(UX) 및 인터페이스
- **유저 프로필 모달**: 사이드바의 설정 아이콘을 통해 자신의 상세 학적 정보를 조회할 수 있는 기능을 구현했습니다.
- **UI 번역 및 최적화**: 비밀번호 찾기 등 주요 안내 메시지를 영어로 번역하고 모달 디자인을 정교화했습니다.

## 4. 데이터베이스 및 기술 문서화
- **실데이터 동기화**: `600여 건`의 학생 데이터를 포함하는 `login_info.json` 및 `user_info.json`을 DB에 완벽히 시딩했습니다.
- **문서 최신화**: `docs/` 폴더 내의 모든 문서와 루트 `README.md`를 현재 시스템 아키텍처와 기능에 맞게 전면 개정했습니다.
