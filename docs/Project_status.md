# 프로젝트 상태 보고서: UCSI 챗봇 (업데이트: 2026-02-06)

## 1. 프로젝트 개요
- **목표:** Python 3.13 기반의 UCSI 대학교 AI 어시스턴트 개발.
- **특징:** Google Gemini/Gemma 하이브리드 지원, 실시간 데이터 동기화, 사용자 인증 및 보안 강화.
- **현재 상태:** 프론트엔드 및 백엔드가 모든 핵심 기능을 갖춘 고도화된 프로토타입 단계.

## 2. 주요 업데이트 사항 (최신)

### ✅ AI 모델 최적화 및 통합
- **모델 전환:** 최신 오픈 모델인 **Gemma 3 27B**(`gemma-3-27b-it`) 및 `gemini-2.0-flash` 하이브리드 지원.
- **구조 개선:** `app/ai/gemini.py`로 모든 AI 로직을 통합하고 최신 Google GenAI SDK 적용.
- **RAG 파이프라인:** Google Gemini Embeddings를 활용한 벡터 검색 기능 활성화.

### ✅ 백엔드 (FastAPI) 및 인증 고도화
- **프레임워크:** FastAPI (Uvicorn 기반).
- **구조:** 모듈식 아키텍처 (`app/auth`, `app/database`, `app/services`).
- **보안 기능:**
  - JWT 기반 로그인/로그아웃 및 세션 유지.
  - 비밀번호 찾기: 학번/이메일 인증을 통한 즉각적인 조회.
- **엔드포인트:**
  - `POST /auth/login`, `POST /auth/find-password`.
  - `GET /auth/profile/{user_id}`: 상세 학생 정보 조회.
  - `POST /chat`: AI 상담 인터페이스.
  - `GET/POST/DELETE /chat/history`: 유저별 대화 기록 관리.

### ✅ 데이터베이스 및 동기화 고도화
- **대규모 데이터 수용:** 6만 건 이상의 학생, 강의, 전공 정보를 수용할 수 있는 아키텍처 구축.
- **자동 동기화:** `seed_db.py`를 통해 JSON 데이터와 MongoDB 간의 안전한 동기화 및 검증 시스템 구축.
- **컬렉션 구조:** `Account` (인증), `User` (프로필), `Course`, `Major`, `ChatHistory` 체계화.

### ✅ 프론트엔드 UI/UX 개선
- **비밀번호 찾기:** 모달 내 인플레이스 인증 및 처리 로직 강화.
- **접근 제어:** 게스트 사용자 채팅 제한 및 안내 문구 표시.
- **프로필 모달:** 사이드바 설정을 통해 상세 학생 정보 조회 기능 추가.
- **로그아웃 최적화:** 세션 종료 시 즉각적인 UI 초기화 및 보안 강화.

## 3. 구현된 기능 현황

### 🔹 백엔드 (FastAPI)
- **인증:** JWT 기반 로그인/로그아웃 및 비밀번호 초기화/찾기 기능.
- **통신:** 비동기(Async) 처리를 통한 빠른 응답속도 확보.

### 🔹 프론트엔드
- 최신 데이터를 반영한 UI 업데이트, 채팅 이력 저장 및 관리 기능 지원.

## 4. 향후 계획
- **Gemma 3 로컬 실행 최적화**: 서버 자원 및 지연 시간 관리.

- **개인화 서비스**: 학업 이력 기반 장학금/수강 신청 맞춤형 상담.
