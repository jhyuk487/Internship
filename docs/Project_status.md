# 프로젝트 상태 보고서: UCSI 챗봇 (업데이트: 2026-02-06)

## 1. 프로젝트 개요
- **목표**: UCSI 대학교 AI 어시스턴트 개발 (Python 3.13 + FastAPI)
- **현재 상태**: 프론트/백엔드 핵심 기능 구현 완료, 고도화 진행 중

## 2. 주요 업데이트 사항

### ✅ AI 모델 및 RAG 통합
- `google-genai` SDK 기반으로 Gemma 3 (`gemma-3-27b-it`) 사용
- FAISS + Google Embeddings 기반 벡터 검색 활성화
- `/chat/ingest`로 문서 인덱스 재생성 가능

### ✅ 인증 및 보안
- JWT 기반 로그인 및 세션 복구
- 비밀번호 찾기 기능 (학번 + 이메일)
- 게스트 사용자 기능 제한

### ✅ 채팅 히스토리
- `/chat/history` API로 저장/로드/핀/삭제 지원

### ✅ 프론트엔드 기능
- GPA 계산기 (학기별 편집 및 로컬 저장)
- 코스 자동완성 검색
- 프로필 모달 및 세션 UI 업데이트

## 3. 구현된 기능 현황
- AI 채팅 (일반/개인 질문 분리)
- 사용자 인증 및 프로필 조회
- 채팅 히스토리 관리
- 코스 검색 및 GPA 계산

## 4. 향후 계획
- RAG 문서 소스 확장
- 개인화 추천 로직 고도화
- 배포 환경용 API 경로 정리
