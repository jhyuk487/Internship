# 테스트 체크리스트

아래 항목은 최근 변경사항 기준으로 테스트가 필요한 부분을 정리했습니다. 테스트 완료 시 체크하거나 삭제하세요.

## 1. 게스트 채팅 (로그인 없이)
- [ ] 로그인 없이 채팅 입력창이 활성화되어 입력/전송 가능
- [ ] 로그인 없이 채팅 응답이 정상 출력됨
- [ ] 개인/민감 정보 질문 시 "로그인 필요" 안내가 표시됨
- [ ] 일반 질문은 정상 답변됨

## 2. 로그인 후 개인정보 접근
- [ ] 로그인 후 프로필 모달 열림
- [ ] 프로필 모달 데이터가 본인 정보로 정상 로드됨
- [ ] 로그인 없이 프로필 모달 접근 시 로그인 요구 모달이 뜸

## 3. 채팅 히스토리 (로그인 필요)
- [ ] 로그인 후 기존 채팅 히스토리 로드됨
- [ ] 채팅 저장/핀/삭제 정상 동작
- [ ] 로그아웃 시 히스토리 UI가 게스트 안내로 바뀜

## 4. GPA 계산기 (로그인 필요)
- [ ] 게스트에서 GPA 모달 접근 시 로그인 요구
- [ ] 로그인 후 GPA 모달 열림 및 기록 저장 정상

## 5. 보안/차단 확인
- [ ] `/auth/profile/{user_id}` 직접 호출 시 토큰 없으면 401/403
- [ ] 토큰이 있어도 타인 `user_id` 요청은 403
- [ ] `/chat` 요청 시 토큰 없으면 개인 정보 조회가 차단됨

## 6. 회귀 확인 (UI)
- [ ] 채팅 영역/버튼/모달 UI 깨짐 없음
- [ ] 다크모드 토글 정상 동작
