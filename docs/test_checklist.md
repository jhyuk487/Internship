# 테스트 체크리스트

아래 항목은 현재 동작 기준으로 테스트가 필요한 부분을 정리했습니다. 테스트 완료 시 체크하거나 삭제하세요.

## 1. 게스트 채팅
- `ALLOW_GUEST_CHAT = true`일 때 입력/전송이 가능함
- 게스트 대화가 `localStorage` 키 `guestChatHistory`에 저장/복원됨
- 개인/민감 정보 질문 시 로그인 필요 안내 메시지가 표시됨

## 2. 로그인 후 개인정보
- 로그인 후 설정(Settings) 클릭 시 프로필 모달이 열림
- 로그인 전 Settings 클릭 시 로그인 요구 모달이 뜸
- `/auth/profile/{user_id}`는 서버에서 토큰 없이 응답함 (현재 동작 확인)

## 3. 채팅 히스토리 (로그인 필요)
- 로그인 후 기존 채팅 히스토리 로드됨
- 채팅 저장/핀/삭제 정상 동작
- 로그아웃 시 히스토리 UI가 게스트 안내로 전환됨

## 4. GPA 계산기 (로그인 필요)
- 게스트에서 GPA 모달 접근 시 로그인 요구
- 로그인 후 GPA 모달 열림 및 `/grades/me` 저장/로드 정상 동작

## 5. 인증/보안 확인
- `/auth/me` 토큰 없으면 401
- `/chat/history`, `/grades/me` 토큰 없으면 401
- `/auth/profile/{user_id}`는 공개 상태 (정책 재검토 필요)

## 6. 회귀 확인 (UI)
- 채팅 영역/버튼/모달 UI 깨짐 없음
- 다크모드 토글 정상 동작 (존재 시)
